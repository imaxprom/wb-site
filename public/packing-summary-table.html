Как читать:
• Левая колонка — «ручка» артикула: имя, кол-во размеров/цветов, артикул WB
• На каждый артикул — N строк по размерам + строка «образец» (жёлтым курсивом, редактируется вручную)
• Всего на складе — редактируемое поле (пунктирная рамка), вводишь сам сколько у тебя есть на складе
• Регионы идут парами: Коробов (редактируется) → Штук (автоподсчёт = коробов × perBox)
• ✎ — ячейка с ручным override (отличается от авторасчёта)
• Отгружено = Σ коробов по всем регионам (включая образец)
• Сверка = Всего на складе − Отгружено. +зелёный = всё ок, −красный = недостаёт товара
• Итоговая строка внизу: по каждому региону и финальная сверка

Данные — мок. В реальности регионы берутся из `useEffectiveRegions()`, артикулы и размеры — из расчёта V3.
